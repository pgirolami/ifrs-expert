# Experiments

This directory contains ad hoc evaluation harnesses, archived experiment runs, and Promptfoo-related artifacts.

For Promptfoo workflow, storage layout, and operational commands, see:
- [`docs/PROMPTFOO.md`](../docs/PROMPTFOO.md)

The main convention to remember is that Promptfoo history is stored per experiment under:

```text
experiments/<experiment_subdir>/.promptfoo/
```
